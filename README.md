### Installation

```bash
# Clone the repository
git clone https://github.com/Onkar86/FlowSpeak.git
cd FlowSpeak

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will open at `http://localhost:5173`

### Build for Production

```bash
# Create production build
npm run build

# Preview production build
npm run preview
```

## ðŸ’¡ Usage

### Voice Recording
1. Click the microphone button or press **Shift+M**
2. Start speaking - your words appear in the editor
3. Click again or press **Shift+M** to stop

### AI Tools
1. Select text in the editor (or have text present)
2. Click any AI tool card
3. Watch your text transform!

### Saving Notes
1. Click the **Save** button when you have text
2. Access saved notes from the Notes tab
3. Delete notes by clicking the trash icon
